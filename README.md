# calc-IIT-gaussian-phi-for-jsai2026
Barrett-Seth-type IIT phi calculation scripts for JSAI2026 presentation.

The original paper of calculation methods: [Barrett, Adam B., and Anil K. Seth. 2011. “Practical Measures of Integrated Information for Time-Series Data.” PLoS Computational Biology 7 (1): e1001052.](https://journals.plos.org/ploscompbiol/article?id=10.1371/journal.pcbi.1001052)
